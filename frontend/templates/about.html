About ShopLocal

What is ShopLocal?

ShopLocal is a simple microservices e-commerce app built for learning DevOps concepts like Docker, Kubernetes, ArgoCD, and CI/CD pipelines.

Tech Stack

Python Flask Docker Kubernetes Helm ArgoCD GitHub Actions Terraform Floci

Architecture

The app is split into 2 microservices — a Frontend service that renders the UI and a Products service that provides product data via REST API. Both run as separate pods inside a local Kubernetes cluster.
